Pandoc

A universal document converter.

Pandoc home page Pandoc main documentation Pandoc MacOS installer Pandoc Windows installer Pandoc Linux installer

Pandoc is a command-line program that will convert any document type to any other document type, like a Markdown document to a Microsoft Word document. It is an especially useful bridge between the flexible comfort of drafting a document in Markdown but producing a handsome pdf via TeX with appropriate formatting, citation management, footnotes, and so on.
