RStudio Desktop

RStudio is an integrated development environment (IDE) for R.

RStudio Desktop home page RStudio Desktop main documentation RStudio Desktop MacOS installer RStudio Desktop Windows installer RStudio Desktop Linux installer

RStudio Desktop provides a complete environment for R programming, including a console, dedicated editor, and other windows for keeping track of vectors, files, and plots. With Pandoc, it can make for an entire scholarly environment for a quantitatively minded social scientist.
